== Description ==

A WordPress plugin that encapsulates the NCS e-Giving platform for use in client WordPress based websites. This plugin will provide e-giving capabilities for anonymous users as well as users with an e-giving account with NCS. Authenticated users may manage some aspects of their e-giving account, schedule offerings, etc.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php do_action('plugin_name_hook'); ?>` in your templates
